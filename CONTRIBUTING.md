# Contributing to prompt-architect

Thank you for your interest in contributing to prompt-architect!

## Getting Started

1. Fork the repository
2. 2. Clone your fork: `git clone https://github.com/your-username/prompt-architect.git`
   3. 3. Install dependencies: `npm install`
      4. 4. Create a feature branch: `git checkout -b feature/your-feature-name`
        
         5. ## Development Workflow
        
         6. - Run the development server: `npm run dev`
            - - Run linting: `npm run lint`
              - - Run tests: `npm run test`
                - - Build for production: `npm run build`
                 
                  - ## Code Style
                 
                  - - This project uses ESLint for JavaScript/TypeScript linting
                    - - Follow the existing code style and formatting conventions
                      - - Ensure all lint checks pass before submitting a PR
                       
                        - ## Pull Request Process
                       
                        - 1. Ensure your branch is up to date with `main`
                          2. 2. Run all checks locally before pushing
                             3. 3. Write clear, descriptive commit messages
                                4. 4. Provide a summary of changes in the PR description
                                   5. 5. Link any related issues
                                     
                                      6. ## Reporting Issues
                                     
                                      7. Please use GitHub Issues to report bugs or request features. Include:
                                      8. - A clear description of the problem
                                         - - Steps to reproduce
                                           - - Expected vs actual behaviour
                                             - - Environment details (OS, Node version, browser)
                                              
                                               - ## Code of Conduct
                                              
                                               - Be respectful and inclusive. Harassment or discrimination of any kind will not be tolerated.
